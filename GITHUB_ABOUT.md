# GitHub 仓库简介

## 300 字以内简介

SIU（Should I Upgrade?）是一个 AI 版本升级审计工具。输入 GitHub 仓库和当前版本，它会读取当前版本到最新正式 Release 之间的所有更新，结合项目简介、README 和 Release Notes，自动总结核心功能、关键修复、破坏性变更，并给出“是否应该升级”的明确建议。

## English Description

AI-powered release gap auditor that compares your current version with the latest GitHub release and answers: Should I upgrade?
